Dummrumleuchte
==============

Activates a rotating light to attract attention when a corresponding CAN message is received.

Idea based on https://wiki.openlab-augsburg.de/Dummrumleuchte .

To build this type "make" and "make flash" to flash the micro controller using avrdude
